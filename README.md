Install linux
-------

git clone https://github.com/bowscoin/bowscoin.git

Bowscoind:

    cd bowscoin/src
    make -f makefile.unix USE_UPNP=- USE_IPV6=-

Bowscoin-qt:

    cd bowscoin
    qmake-qt4
    make
